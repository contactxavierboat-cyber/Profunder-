2.5X EXPOSURE LOGIC

Calculate:

Total Current Exposure =
All revolving limits + business lines + active installment credit.

Projected Exposure Ceiling =
Total Exposure × 2.5

Adjust multiplier:

If utilization > 40% → max 1.8x
If inquiries > 5 in 90 days → max 1.5x
If recent derogatory (12 months) → max 1.25x
If score < 660 → cap at 1.5x
If strong profile → allow up to 3.0x

Label clearly:

“Projected Exposure Ceiling Based on Current Profile Strength.”

Never guarantee approvals.
